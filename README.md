# Anima Canvas Suite

A modern travel/experiences landing page built with:

- React + TypeScript
- Vite
- Tailwind CSS & shadcn/ui
- Radix UI primitives

## Development

Install dependencies:

```bash
npm install
# or
bun install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
